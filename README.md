## AYR Bot Facebook
PHP Curl Facebook Untuk Termux

## Tutorial Install
```bash
* pkg update && upgrade
* pkg install git
* pkg install php
* git clone https://github.com/amirulcorp/ayrbotfb
```

## Tutorial Menjalankan Bot
```
* cd ayrbotfb
* cd run
* php br.php
```
